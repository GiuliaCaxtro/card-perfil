
#  Card de Perfil - Giulia Castro

Este é um projeto de **card de perfil profissional**, desenvolvido com HTML e CSS, com foco em design moderno, organização de conteúdo e responsividade.

🔗 Acesse o projeto online:
👉 https://giuliacaxtro.github.io/card-perfil/

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts

---

## 🎯 Funcionalidades

* Exibição de foto de perfil
* Nome e profissão
* Links para redes sociais (LinkedIn, GitHub e Instagram)
* Seção "Sobre mim"
* Botão de contato por e-mail
* Layout moderno com efeito glassmorphism
* Estrutura organizada e limpa

---

## 🎨 Design

O projeto utiliza:

* Gradiente de fundo moderno
* Card dividido em duas seções
* Bordas arredondadas personalizadas
* Sombras suaves
* Tipografia com Google Fonts

---

## 📁 Estrutura do projeto

```
📦 card-perfil
 ┣ 📜 index.html
 ┣ 📜 style.css
 ┗ 📂 imagens
    ┣ 📷 perfil02.png
    ┣ 📷 icons (redes sociais)
    ┗ 📷 favicon
```

---

## 💻 Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/GiuliaCaxtro/card-perfil.git
```

2. Abra o arquivo `index.html` no navegador.
